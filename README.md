# 游戏动作批量裁剪工具
专为游戏动作参考视频设计的一键批量裁剪工具，完全符合你1080p/30fps的输出要求，零代码基础就能用。

## 📦 下载地址（直接下完整包）
最新版完整工具包直链：https://github.com/terrence2025/action-cutter-tool/archive/refs/heads/main.zip

## 🚀 使用步骤（3步搞定）
1. **补全FFmpeg依赖**：先下载[FFmpeg静态包](https://github.com/BtbN/FFmpeg-Builds/releases/download/latest/ffmpeg-master-latest-win64-gpl.zip)，解压后把里面的`ffmpeg.exe`放到工具根目录和`一键打包.bat`同文件夹
2. **一键打包exe**：双击运行`一键打包.bat`，等待1分钟自动生成独立exe文件，存到`dist`目录
3. **开始使用**：双击`dist`文件夹里的`动作裁剪工具_GUI.exe`即可打开图形化工具，选择视频文件夹→选全自动模式→开始处理，全程不用写任何命令

## ✨ 功能特性
- ✅ 自动去掉视频开头/结尾空帧、黑屏
- ✅ 自动识别3次动作循环，只保留1个完整无重复的动作
- ✅ 输出默认就是1080p/30fps mp4，画质无损
- ✅ 批量处理，一次可以裁剪几百个视频
- ✅ 原视频不会被修改，自动存到「裁剪完成」目录

## 📝 参数说明
- 空帧检测阈值：默认0.1就够用，空帧识别不准就调大到0.2
- 动作循环个数：你每个动作录了3次，默认选3就行
